Introdução ao Desenvolvimento Web

Sempre sonhou em criar seu próprio site ou jogo online? Este curso é o seu primeiro passo! Aprenda as linguagens fundamentais do desenvolvimento web: HTML, CSS e JavaScript. Você poderá estruturar páginas da web, adicionar estilos visuais e até torná-las interativas.

Introdução ao Desenvolvimento Web
(6 semanas, 120 horas)

Este curso apresenta os blocos de construção fundamentais da web, incluindo HTML, CSS e JavaScript. Você aprenderá a estruturar páginas da web, formatar conteúdo e adicionar interatividade usando código. Este curso é uma ótima base para estudos posteriores de desenvolvimento web ou um ponto de partida para uma carreira em web design.

Noções básicas de segurança cibernética

Este curso explora o mundo em constante evolução da segurança cibernética. Você aprenderá sobre ameaças comuns como phishing e malware, bem como estratégias defensivas para proteger seus dados e privacidade online. O curso aborda as melhores práticas para gerenciamento de senha, navegação segura e reconhecimento de golpes online.

Noções básicas de segurança cibernética
(8 semanas, 160 horas)

A internet é um lugar vasto e incrível, mas é importante saber como se manter seguro. Este curso irá equipá-lo com o conhecimento para se defender contra ameaças online comuns. Aprenda sobre golpes de phishing, malware e como proteger seus dados e privacidade.

Modelagem e animação 3D

Tem paixão por criar personagens e mundos fantásticos? Neste curso, você aprenderá as habilidades para fazer isso acontecer no mundo digital. Explore o software de modelagem 3D, aprenda a aplicar texturas e iluminação e dê vida às suas criações por meio de técnicas de animação.

Modelagem e animação 3D
(12 semanas, 240 horas)

Este curso mergulha no emocionante mundo da modelagem e animação 3D. Você aprenderá a criar modelos 3D usando software padrão da indústria, aplicar texturas e iluminação e dar vida às suas criações por meio de técnicas de animação. Este curso é um ótimo ajuste para qualquer pessoa interessada em efeitos visuais, jogos ou campos de animação.

Introdução à Programação

Já se perguntou como os programas de computador são criados? Este curso oferece um gostinho do emocionante mundo da codificação. Aprenda conceitos fundamentais de programação como variáveis, tipos de dados e fluxo de controle usando uma linguagem amigável para iniciantes. Ao final, você poderá escrever programas básicos e entender como os programadores pensam.

Introdução à Programação
(10 semanas, 200 horas)

Este curso fornece uma base sólida em fundamentos de programação. Você explorará conceitos básicos de programação como variáveis, tipos de dados, fluxo de controle e funções usando uma linguagem de programação amigável para iniciantes como Python. Ao concluir, você será capaz de escrever programas básicos e terá uma sólida compreensão da lógica de programação.

Fundamentos de Rede de Computadores

A internet conecta pessoas em todo o mundo, mas como tudo funciona? Este curso mergulha nos blocos de construção de redes de computadores. Aprenda sobre componentes essenciais como roteadores e switches, como os dados viajam pela web e a importância da segurança de rede.

Fundamentos de Rede de Computadores
(8 semanas, 160 horas)

Este curso explora os blocos de construção de redes de computadores. Você aprenderá sobre componentes de rede como roteadores e switches, protocolos de transmissão de dados e conceitos de segurança de rede. O curso fornece uma base para estudos posteriores em administração de rede ou carreiras de suporte de TI.

Introdução à Ciência de Dados

Os dados estão por toda parte, mas como você faz sentido de tudo isso? Este curso apresenta o mundo da ciência de dados. Aprenda como coletar dados, analisá-los usando ferramentas úteis e criar visualizações para contar uma história com as informações que você encontrar.

Introdução à Ciência de Dados
(12 semanas, 240 horas)

Este curso oferece uma introdução ao emocionante campo da ciência de dados. Você aprenderá sobre coleta de dados, técnicas de análise e ferramentas de visualização de dados, como planilhas e programação básica. O curso ajuda a desenvolver habilidades de pensamento crítico para trabalhar com dados e prepara você para estudos posteriores em análise de dados ou ciência de dados.
